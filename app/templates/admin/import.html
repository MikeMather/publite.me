Supported Frontmatter Fields

  • title - Post title
  • slug - URL slug (auto-generated if not provided)
  • date - Publication date (YYYY-MM-DD format)
  • published - Whether post is published (true/false)
  • page - Whether this is a page (true/false)
  • tags - Comma-separated tags
  • description - Meta description
  • keywords - Meta keywords
  • canonical_url - Canonical URL
  • og_image - Open Graph image
  • no_index - Whether to exclude from search engines (true/false)

Example Frontmatter

title: My Awesome Post
date: 2024-01-15
published: true
tags: blog, tutorial, web development
description: A great post about web development
---
